# Actualadam Tap

## How do I install these formulae?

`brew install actualadam/tap/<formula>`

Or `brew tap actualadam/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "actualadam/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
